# Módulo Preinscripción y Admisiones

Este módulo maneja el flujo de ingreso de un `[[Adulto_Mayor]]` al sistema.

## Proceso de Admisión
1. Los responsables llenan un formulario de preinscripción.
2. Se generan notificaciones automáticas.
3. Se programan citaciones para entrevistas diagnósticas (con el Trabajador Social y el Médico).
4. Asignación definitiva del adulto mayor a una categoría de servicio (Independiente, Compartida o Diurna).
